# Competitive Landscape

## Market Map

DevStage operates at the intersection of four categories, but no competitor
occupies the same position:

```
                    Real Work
                       |
        Automattic     |    DevStage
        Linear         |    (UNIQUE)
        Zapier         |
                       |
  One-Way  -----------+----------- Bidirectional
                       |
        HackerRank     |    Hired
        CodeSignal     |    Wellfound
        Devskiller     |
                       |
                  Assessment Only
```

### The Quadrants

| Quadrant | Players | What's Missing |
|----------|---------|----------------|
| Real Work + One-Way | Automattic, Linear, Zapier | Internal processes, not platforms; only company evaluates |
| Assessment + One-Way | HackerRank, CodeSignal | Simulated tests, not real work; developer has no agency |
| Assessment + Bidirectional | Hired, Wellfound | Profile-based matching, no actual work involved |
| **Real Work + Bidirectional** | **DevStage (alone)** | -- |

## DevStage's Three Unique Differentiators

No competitor combines all three:

1. **Bidirectional listings** -- both companies AND developers can post
2. **Real project work** -- not simulated tests, not interviews
3. **Structured outcome reviews** -- pass/partial/fail per measurable criterion

## Key Risk: Mercor

Mercor raised **$100M at $2B valuation** (February 2025) pursuing AI-native hiring.
They use AI to screen and match candidates. However, they do NOT offer:
- Developer-posted listings
- Real project work (it's still interview-based)
- Structured outcome criteria
