# Pain Points

The problems DevStage solves, backed by data. These pain points drive the entire
value proposition and landing page messaging.

## Severity Ranking

| Severity | Pain Point | Who Feels It | Key Stat | File |
|----------|-----------|--------------|----------|------|
| Critical | [Hiring Costs](hiring-costs.md) | Companies | $4,700-$36K per hire | Revenue-saving value prop |
| Critical | [Resume Fraud](resume-fraud.md) | Companies | 55% of IT employees lie | Trust/verification value prop |
| Critical | [Experience Paradox](experience-paradox.md) | Developers | 73% entry-level decline | Core developer value prop |
| High | [Bad Hire Costs](bad-hire-costs.md) | Companies | $150K-$300K per bad hire | Risk reduction value prop |
| High | [AI Cheating](ai-cheating-interviews.md) | Companies | 48% candidates cheat | Modern urgency angle |
| High | [Internship Decline](internship-decline.md) | Developers | -30% since 2023 | Pipeline crisis angle |
| Medium | [Interview Inefficiency](interview-inefficiency.md) | Both | 57% accuracy | Process improvement angle |

## How Pain Points Map to Landing Page Sections

- **Hero**: Combines experience paradox (dev) + hiring costs (company)
- **Pain: Companies**: Hiring costs + resume fraud + interview inefficiency
- **Pain: Developers**: Experience paradox + internship decline
- **Solution**: Outcome criteria as answer to all pain points
- **Numbers Section**: Key stats from all pain points
