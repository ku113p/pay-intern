# Target Audiences

DevStage serves 7 distinct audience segments across two sides of the marketplace.
Each segment has unique pain points, motivations, and messaging angles.

## Priority Ranking

| Priority | Segment | Side | Size Signal | File |
|----------|---------|------|-------------|------|
| 1 | [Junior Developers](junior-developers.md) | Developer | 73% entry-level decline, 6.1% CS unemployment | Largest underserved group |
| 2 | [Companies Frustrated with Hiring](companies-hiring.md) | Company | 74% made bad hires, $4,700+ cost-per-hire | Revenue driver |
| 3 | [Career Switchers](career-switchers.md) | Developer | 1 in 3 tech workers came from other fields | Fast-growing segment |
| 4 | [Budget-Constrained Startups](startups-budget.md) | Company | 99.95% startups without VC | Underserved by recruiters |
| 5 | [Upskilling Developers](upskilling-developers.md) | Developer | 69% actively learning new tech | High spending power |
| 6 | [Challenge Seekers](challenge-seekers.md) | Developer | 52% code for fun after work | Viral potential |
| 7 | [Mid-to-Senior Path](mid-to-senior.md) | Developer | 1 in 4 get zero growth opportunities | Premium segment |

## Cross-Segment Insights

- **Developer-side segments outnumber company-side 5:2** -- initial marketing should focus on developer acquisition
- **All developer segments share one core need**: verified, real-world experience that can't be faked
- **Both company segments share one core need**: evaluating developers through real work, not resumes
- **Challenge seekers and upskilling developers overlap significantly** -- can be targeted with shared campaigns
