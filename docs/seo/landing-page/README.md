# Landing Page Strategy

## Overview

A single, long-scroll, high-converting promo page at `/promo` with its own unique design
(not using the shared Layout component). The page targets all 7 audience segments
with a unified narrative flow.

## Core Principle

**Pain -> Solution -> Proof -> Action**

Every section follows this arc. The page progressively builds the case from
"hiring is broken" to "here's how DevStage fixes it" to "here's the proof" to
"get started now."

## Key Decisions

1. **Unified page** (not separate pages per audience) -- visitor self-selects through CTAs
2. **Company pain first** (sections 2, 5) then developer pain (sections 3, 6) -- revenue side first
3. **Outcome criteria showcase** (section 7) -- core feature gets its own section
4. **Challenge seekers** (section 8) -- captures the "fun" angle
5. **Comparison table** (section 10) -- positions against alternatives
6. **8+ CTA buttons** -- every scroll stop has a next action
